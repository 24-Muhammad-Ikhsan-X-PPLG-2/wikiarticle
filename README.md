✨ Dibuat untuk:

- 📖 Membaca artikel dengan nyaman

## 📦 Installation

### 1️⃣ Clone repository

```bash
git clone https://github.com/24-Muhammad-Ikhsan-X-PPLG-2/wikiarticle.git
cd wikiarticle
```

### 2️⃣ Install dependencies

```bash
bun install
```

### 3️⃣ Run development

```bash
bun run dev
```

🚀 Buka di browser:

```
http://localhost:3000
```
